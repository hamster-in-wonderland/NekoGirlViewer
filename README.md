# NekoViewer
一个从nekos.bestAPI上浏览和下载猫娘图片的工具
喵~

## 介绍
从nekos.bestAPI上浏览和下载猫娘图片
通过pyinstaller编译

灵感来源于NekoGirl

https://github.com/YUZU384/NekoGirl-winui3

## 环境要求
运行项目需要的环境/软件
- Python 3.8+

## 快速运行
拉取代码
```bash
git clone https://github.com/hamster-in-wonderland/NekoGirlViewer/
```
下载支持库：
```bash
PySide6 requests
```
